# CHECKMATISSE · Шах и Матисс

Рогалик-шахматы с картами в стиле аппликаций Матисса и супрематизма Малевича.
Масляная живопись, стоп-моушен анимация, бумажный коллаж.

**Жанр:** roguelike deck-builder на шахматном поле перед замком.
**Режимы:** соло против поискового ИИ · сетевой кооп с другом (общая армия) против ИИ.

## Как играть

- Забег по карте пути: бои, элитки, события, лавки, привалы → босс «Осада Замка».
- Бой: шахматные ходы + карты за «краску» (копится от взятий).
- Карты режут доску ножницами, клеят новые клетки, призывают, замораживают, толкают в дыры.
- Армия персистентна: погибшие фигуры не возвращаются. Пешки мутируют при промоушене
  (Голубь Матисса, Красный Танцор, Чёрный Квадрат…).
- Куратор периодически сдвигает ряды коллажа — следи за телеграфом.

## Разработка

```bash
npm install
npm run dev        # http://localhost:5173
npm test           # vitest: движок, ИИ, мета
npm run e2e        # playwright smoke
npm run build      # прод-сборка (dist/)
```

## Архитектура

- `src/engine` — чистый детерминированный движок (lockstep-ready, make/unmake для ИИ)
- `src/content` — фигуры/карты/черты/реликвии/энкаунтеры как данные
- `src/ai` — alpha-beta в Web Worker
- `src/render` — PixiJS 8: доска, стоп-моушен 12fps, карты, частицы
- `src/diorama` — Three.js: диорама замка на заднем плане
- `src/meta` — забег: карта пути, лавки, события, сейвы
- `src/net` — кооп через Supabase Realtime (host-authoritative lockstep)

Арт-пайплайн: `scripts/art-prompts.md`.
