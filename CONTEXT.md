## Что это

**CHECKMATISSE** («Шах и Матисс») — рогалик-шахматы с картами в стиле аппликаций
Матисса и супрематизма Малевича. Масляная гуашь, стоп-моушен 12 fps, бумажный коллаж.
Прод: **https://srandagame.vercel.app/** (Vercel, автодеплой из git).
Репозиторий: `arniByte/srandagame`, рабочая ветка `claude/roguelike-matisse-game-gqgpkc`.

**Геймплей**: забег по карте пути (Slay the Spire) → шахматные бои на досках с дырами
перед замком → карты за ресурс «краска» (копится от взятий): ножницы режут клетки,
клей клеит новые, толчок сталкивает в дыры; Куратор сдвигает ряды коллажа;
пешки мутируют при промоушене (Голубь/Танцор/Квадрат/Лоза); армия персистентна —
погибшие фигуры теряются насовсем. Соло против поискового ИИ + сетевой кооп
(общая армия, ходы по очереди: нечётные — хост, чётные — гость).

## Архитектура (карта модулей)

- `src/engine/` — **чистый детерминированный движок** (lockstep-ready): только целые
  числа, RNG xoshiro128** в состоянии, никакого Math.random/Date. Два API:
  `apply()` (чистый) и `applyMut()/unmake()` (make/unmake для ИИ). Редьюсер
  возвращает `EngineEvent[]` — рендер/звук/диорама слушают через шину.
  Взятие короля = победа (мата нет). Ворота с HP бьются бамп-атакой.
- `src/content/` — фигуры/карты/черты/реликвии/энкаунтеры как данные.
  **Порядок регистрации фиксирован** (индексы Zobrist) — новое добавлять в конец.
  `loadContent()` обязателен на буте (main, worker, тесты).
- `src/ai/` — alpha-beta + iterative deepening + TT (Zobrist) в **Web Worker**.
  Негамакс с нюансом: знак меняется только при смене active (несколько действий
  за ход). Карты в поиске: эвристические кандидаты ≤6/узел, бюджет 2 на линию.
  Тиры: apprentice 300мс / journeyman 900мс / master 2500мс.
- `src/meta/` — RunState (чистые редьюсеры), генерация карты акта (13×7, 5 путей),
  лавка/события/награды, сейвы в localStorage (`cm.run.v1`), мид-битва = сид+лог.
- `src/net/` — кооп: host-авторитарный lockstep через **Supabase Realtime**
  (broadcast+presence, канал `cm:{код}`), hash-сверка на каждом действии,
  авто-ресинк снапшотом (gzip). Транспорт-абстракция + loopback для тестов.
- `src/game/controller.ts` — оркестратор: бой↔мета↔ИИ↔сцена↔сеть, экраны.
- `src/render/` — PixiJS 8: перспективная трапеция доски (`board/projection.ts`,
  PERSP=0.36), стоп-моушен твинер с квантованием 12 fps (`anim/`), рука карт
  с drag-пружиной, брызги краски, коллажные переходы. Правило: **доска = 12 fps,
  UI = 60 fps**. Параллакс: слои сцены + глубинный сдвиг рядов через **pivot**
  (не конфликтует с твинами позиций) + «рябь» плиток под курсором.
- `src/diorama/` — Three.js диорама (слои: небо/солнце/холмы/замок/луг + затемнение
  0.34), параллакс за мышью, реакции на события боя, пролетающий голубь.
  Сгенерённый арт грузится по фиксированным путям `/assets/bg/*.webp` с фолбэком
  на canvas2d-рисование.
- `src/ui/` — Preact DOM-оверлей (меню/карта/лавка/событие/привал/награда/лобби).
- `src/assets/manifest.ts` — единая точка ассетов: `assets.texture(key)`;
  нет в манифесте → кодовый супрематический плейсхолдер. `public/assets/manifest.json`
  с `pivot`/`worldScale` на фигуру.
- Слои страницы: `#diorama` (three) → `#game` (pixi, ВЕСЬ ввод) → виньетка → `#ui` (DOM).

## ⚠️ Грабли, на которые уже наступали (НЕ ПОВТОРЯТЬ)

1. **`#ui * { pointer-events: auto }` перекрывал канвас** — полноэкранные
   DOM-экраны съедали все клики. Теперь: `#ui { pointer-events: none }`,
   интерактивность включают классы в styles.css (`.screen` auto,
   `.screen.transparent` none + дети auto).
2. **`app.stage.hitArea` (весь экран) ГЛУШИТ hit-testing детей в Pixi** — из-за
   этого не работали ни клики по доске, ни drag карт. Глобальное слежение
   за курсором для драга — только через DOM `window.addEventListener`.
3. Регрессию ввода ловит `tests/e2e/realInput.spec.ts` (НАСТОЯЩАЯ мышь) — не удалять.
4. Node `fetch` НЕ ходит через прокси окружения — для скачивания использовать
   `curl` (уважает HTTPS_PROXY и системный CA).
5. Chromium+прокси окружения молча вешает TLS — браузерные тесты с внешней сетью
   в этом контейнере не работают (кооп проверен: протокол loopback-тестами,
   транспорт — Node-клиентами Supabase через undici EnvHttpProxyAgent).
6. Шрифты (Amatic SC/Neucha/Caveat, кириллица, @fontsource) должны загрузиться
   ДО создания Pixi Text — см. await в `main.tsx`.

## Пайплайн арта (Higgsfield MCP)

- Стайл-байбл: `scripts/art-prompts.md` (палитра, префиксы промптов).
- Модель фигур/UI: `recraft_v4_1` (1.25 кредита/шт, `colors` + `background_color`),
  вырезание: `remove_background`. Лимит: 8 одновременных задач.
- Конвейер: `scripts/process-assets.mjs <spec.json>` — curl-скачивание → кроп
  (альфа-bbox или фикс. `crop` [x0,y0,x1,y1] в долях) → ресайз → webp →
  фрагмент манифеста в stdout. Виды: piece(h512)/card(w320)/wide(w1600)/strip.
- Фигуры: worldScale = 144/H, ширина спрайта ≤1.06 клетки (см. историю правок
  манифеста), pivot [0.5, 0.97].
- Генерации живут в истории Higgsfield аккаунта; URL-паттерн:
  `https://d8j0ntlcm91z4.cloudfront.net/user_.../hf_ДАТА_ВРЕМЯ_jobid.png`
  (точный URL брать из show_generations/job_display, время ≠ времени запроса!).
- Баланс: ~750 кредитов (план Plus).
